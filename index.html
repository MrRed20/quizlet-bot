quizlet-bot

A bot that automatically calculates and submits answers for Quizlet Live. For... educational purposes only... obviously...

How To Use

Bookmarklet

Create a bookmark with the following url:

javascript:qb_script=document.createElement("script");qb_script.src="https://pandapip1.github.io/quizlet-bot/bot.js";document.head.appendChild(qb_script);

Alternatively, drag this to your bookmarks bar: Quizlet Bot

Create Bookmark

After you enter the code for a quizlet live game, but before it starts, click the bookmark to load the bot.

Tampermonkey/Greasemonkey/etc

Install the userscript manager for your browser:

  • Chrome: Tampermonkey
  • Firefox: Greasemonkey

After you've installed the extension, go to the userscript source here and click install.
